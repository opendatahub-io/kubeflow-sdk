Contributing
============

We welcome contributions! The Kubeflow SDK is a community project under active development.

Getting Started
---------------

1. Fork the `repository <https://github.com/kubeflow/sdk>`_
2. Read the `Contributing Guide <https://github.com/kubeflow/sdk/blob/main/CONTRIBUTING.md>`_
3. Pick an `issue <https://github.com/kubeflow/sdk/issues>`_ or propose a new feature

Community
---------

- **Slack**: `#kubeflow-ml-experience <https://www.kubeflow.org/docs/about/community/#kubeflow-slack-channels>`_
- **Meetings**: `Kubeflow SDK bi-weekly meetings <https://bit.ly/kf-ml-experience>`_
- **GitHub**: `kubeflow/sdk <https://github.com/kubeflow/sdk>`_
